3. Thống kê mô tả dữ liệu với DataFrame.describe()¶

  • a ra bng thng k m t (Descriptive Statistical) ca d liu.
  • Nu d liu l s (numeric) --> Bng thng k s c: count, mean, std, min, max,...
  • Nu d liu l ch (string) --> Bng thng k s c: count, unique, top, freq

Mt lu khi dng describe:

  • To call describe() on just the numerical columns use describe(include = [np.number])
  • To call describe() on just the objects (strings) using describe(include = ['O'])
  • Ref: https://stackoverflow.com/questions/24524104/pandas-describe-is-not-returning-summary-of-all-columns

Quy trình tạo WordCloud đơn giản¶

  • a ni dung cn to WordCloud vo 1 bin String (ni dung sch, ni dung messages,...).
  • Import cc package cn thit ca WordCloud.
  • Import cc STOPWORDS package tu theo tng loi ngn ng.
  • Chy WordCloud
  • Xem Result ri iu chnh li thng s cho hp l
  • Lu WordCloud result vo file nh

Nu bn thc mc STOPWORDS l g th theo mnh hiu, l nhng t m c thm vo vn bn lm cho bi vn smooth hn, mm mi hn. Thng cc t ny chim kh nhiu ch trong bi ca bn nn bn c th b n i. Bn c th xem thm ti: https://en.wikipedia.org/wiki/Stop_words

STOPWORDS ting Anh mnh s dng default ca WordCloud STOPWORDS ting Vit mnh s t parse file ti http://xltiengviet.wikia.com/wiki/Danh_s%C3%A1ch_stop_word

Guide config WordCloud: https://amueller.github.io/word_cloud/generated/wordcloud.WordCloud.html

Pha trn l quy trnh n gin, by gi l cch lm ca mnh dnh cho vic to WordCloud cho ni dung Feeds ca mnh.
